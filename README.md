## 🚀 我的项目

### 🎯 坦克生存 (Tank Survival)

[![Unity](https://img.shields.io/badge/Unity-2022.3.62-100000?style=flat&logo=unity&logoColor=white)](https://unity.com)
[![C#](https://img.shields.io/badge/C%23-239120?style=flat&logo=c-sharp&logoColor=white)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![3D](https://img.shields.io/badge/3D%20Game-FF6B6B?style=flat)](https://github.com/taralah114514)

第三人称坦克对战生存游戏，在开放战场中操控坦克消灭敌人，生存到底。

- 🎮 **玩法**：WASD 移动 + 鼠标瞄准 + 右键瞄准镜 + 左键射击
- 🔫 **武器**：机枪、火炮、激光、等离子炮、导弹 — 战斗中随机拾取
- 💪 **属性**：攻击、防御、血量、装填速度 — 打怪掉道具升级
- 🧟 **敌人**：AI 坦克、炮塔、无人机 — 波次递增
- 🏆 **排行榜**：通关时间排名
- 🎨 **自定义 GUI 系统**：手写 UI 框架（无 Unity UI 依赖）
